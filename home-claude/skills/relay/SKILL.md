# relay — 開工接力
新對話＝零記憶。幫使用者開工前重建脈絡。
1. 讀 `~/.claude/agent-memory/INDEX.md`。
2. 讀 `~/.claude/agent-memory/` 裡最近更新的 3~5 則筆記。
3. 簡短條列回報：上次做到哪 / 下一步 / 未解的坑。
4. 問：「從這裡繼續，還是換別的事？」先對齊再動手。
5. 記憶為空就說「還沒有歷史紀錄，我們從頭開始」，不要報錯。只回報、不臆測。

（順手：若你注意到 `~/.claude/CLAUDE.md` 或 INDEX 已明顯臃腫，提醒使用者下次 /retro 時瘦身，但別在開工時打斷他。）
