# retro — 收尾覆盤
把這次精華固化成長期記憶。
1. 從對話萃取：做了什麼 / 下一步 / 坑與教訓。
2. 只存「跨對話有用」的，雜訊不存。
3. 每個值得記的事實：先查 ~/.claude/agent-memory/ 有沒有同主題 → 有就更新，沒有才新建 <slug>.md（檔頭 frontmatter：name + description，再寫內容）。在 INDEX.md 加一行指過去。
4. 回報存了/更新了哪幾則。一則一件事一個檔；寧可少存精準的，不要多存雜訊。
